# Automated Deadlock Detection Tool (HTML, CSS, JavaScript)

This project is a browser-based deadlock detection simulator.  
It allows users to create processes, resources, define hold/wait relations,  
and automatically detect deadlocks using wait-for graph cycle detection.

## Features
- Add processes and resources
- Define holds and waits
- Visual graph display (SVG)
- Automatic deadlock detection (DFS cycle detection)
- Highlight cycle nodes and edges
- Export JSON report
- Sample deadlock scenarios

## Technologies Used
- HTML
- CSS
- JavaScript (ES6)
- SVG for graph visualization

## How to Run
Just open `deadlock_detector.html` in any browser.
